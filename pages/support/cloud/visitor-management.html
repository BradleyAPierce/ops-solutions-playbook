School Gate Guardian (SGG) is a visitor management check-in solution. The software provides instantaneous sex-offender background checks, emergency mass alerts, and web based reporting among other notable features. Their add-on solutions include Volunteer Management Check-In, Attendance Check-In, RSVP, and Digital Hall Pass solutions.

The visitor management software, is now available on-premises or as a cloud-hosted solution. This hybrid software as a Service (SAAS) solution gives you the flexibility to host data on your own network or in a secured hosted environment.

School Gate Guardian's Hosted Solution maintains multiple layers of security, multiple internet connectivity, backup and power.

SECURITY FEATURES of the Hosted Solution:
• Security starts with clustered firewalls to help filter out unwanted traffic
• Network monitoring watches for anything odd and alerts hosting administrators
• All data sent to and from servers are sent via an encrypted 256-bit SSL (TLS 1.2) channel
• Servers are scanned and locked down according to PCI DSS 3.1
• Data communication between internal web service and database servers are encrypted via a separate 256-bit SSL (TLS 1.2) channel
• Database data is encrypted at rest, meaning your data is always protected
• All TLS certificates are constantly, and automatically, kept up to date to make sure
communications are always using the latest encryption
• Third party environmental and security monitoring
• Each customer has their own database and web services, and are secured with their own authentication
• Multiple fields are required for authentication to our Hosted service and are unique to each customer
• For any transfer of bulk data between School Gate Guardian (or your SIS), SGG implements SFTP to keep the file transfers secure. API integration occurring with some SIS systems.

The School Gate Guardian hosting facility performs daily backups with copies stored off site, maintains multiple Tier 1 internet connections, independent cooling, direct grid power with UPS and automatic generator fail-over. All services are on a highly available service to automatically fail-over to other hardware in case of an issue.
