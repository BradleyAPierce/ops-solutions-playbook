Example Page with Custom Header

This demonstrates how to customize the header logo per page using data attributes.

How This Works

  1. The <div id="header-container"> has data-logo and data-subtitle attributes
  2. The component loader fetches /components/header.html
  3. After loading, it checks for data attributes
  4. If found, it swaps the image src to your custom path

Example Code

<!-- Default header (uses header.html default images) -->
<div id="header-container"></div>

<!-- Custom logo for Healthcare section -->
<div id="header-container" 
     data-logo="/assets/images/logos/healthcare-logo.svg">
</div>

<!-- Custom logo AND subtitle for Finance section -->
<div id="header-container" 
     data-logo="/assets/images/logos/finance-logo.svg"
     data-subtitle="/assets/images/content/finance-subtitle.svg">
</div>
