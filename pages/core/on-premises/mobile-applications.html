bizhub Remote Access App

The bizhub Remote Access App is an easy and convenient app that provides one-to-one functionality with the bizhub MFP for touchless operations. Compatible with both iOS and Android, the bizhub Remote Access app allows users to operate the latest bizhub MFPs from smartphones and tablet devices. Users can operate the keys on the bizhub LCD control panel as well as the hard keys in the same way wirelessly and without touching the MFP Control Panel. Ideal for those that take extra hygiene precautions and for individuals that find it easier to execute operations from their mobile device.

For MFP administrators, settings for each bizhub can be made remotely. bizhub Remote Access can also be very useful for maintenance and staff training. The bizhub Remote Access App provides an additional tool to make operating your MFP seamless and efficient while maintaining a clean work environment

Remote Panel

Allows users to operate the control panel of bizhub device remotely from a computer on the network, allowing users to remotely operate the machine from the computer. The device status can also be remotely monitored. This function promotes efficiency in operation and facilitates training sessions and demonstrations.  Advantages of using Remote Panel include:

  • Operation of the MFP's user interface from any office computer/laptop
  • Provides effective means of demonstrating device operation
  • Ability to remotely train users and configure network and fax settings
  • The web user interface (UI) provides a remote emulation of the device's control panel in which both the hard and soft buttons are displayed and provides operable functionally from the client's workstation
