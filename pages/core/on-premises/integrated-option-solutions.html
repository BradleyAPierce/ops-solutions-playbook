Integrated Option Solutions (i-Option) Kits:

LK-104 VOICE GUIDANCE – Provides verbal guidance to users to explain screens, keys and main functions, which helps users carry out MFP operations accurately. This function also helps people who have difficulty viewing the screen to execute operations with greater ease via voice guidance.

LK-105 OCR TEXT RECOGNITION– Offers full OCR text recognition for PDFs and highly compressed Compact PDF documents, enables users to copy and paste text sections of the file for reuse in other documents and applications — no need to retype the data, reducing margin for error while increasing efficiency.

LK-106 BARCODE FONTS– Detects PCL barcode sequences in print data and prints these as barcodes using bizhub Konica Minolta devices. Supports barcode fonts essential for SAP printing and other print environments that use Unix, DOS or AS400 computer systems. Users can print their files without the need for additional printer software.

LK-107 UNICODE FONTS – Unicode, a worldwide text standard, consists of over one million characters, including almost every existing symbol and character worldwide. By supporting fonts of multiple languages, users can easily print files without encountering problems with illegible characters in the document. The LK-107 ensures that the MFP prints an extensive selection of symbols and characters even within a single print job.

LK-108 OCR A AND B FONTS – Supports two font types that are widely used in banking and financial applications, OCR A and OCR B — differences in shape and style set them apart. OCR-A, standardized by the American National Standards Institute, uses simple, thick strokes with a fixed width to form recognizable characters. OCR-B is ISO-certified and often requested for form or check printing, primarily in the financial sector.

LK-110 DOCUMENT CONVERTER PACK –  Offers Searchable PDF and conversion to Office Open XML File formats (DOCX, XLSX) to enhance work efficiency as well as the ability to print Email attachments, encrypt PDF files, create files that comply with PDF/A ISO 19005 standards (PDF A/1-a & PDF A/1-b) and convert files to Linearized PDF for efficient viewing of files posted on web sites.

LK-111 THINPRINT® CLIENT – ThinPrint® streamlines printing in those environments that have implemented this technology to minimize bandwidth bottlenecks and degradation in network performance. ThinPrint can compress print jobs up to 98%, effectively addressing constant heavy network print traffic. This high compression improves print speed and reduces network traffic, thereby enabling the entire corporate network to work more efficiently.

LK-116 BITDEFENDER ANTIVIRUS –  Activates several MFP security features to ensure a safe operating environment. The LK-116 notifies the user whenever the MFP faces a potential risk from virus-infected files. All incoming and outgoing data such as print jobs, scans, faxes and even encrypted PDF box files are scanned for potential malware and viruses. Also prevents potential risks from outside the company to enter through the MFP.
