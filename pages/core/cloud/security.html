Stratus Bizhub SECURE Notifier Dashboard

Konica Minolta’s Dispatcher Stratus app provides users with seamless access to their scan-to-cloud workflows. This easy-to-use app includes a number of powerful features, such as:

  • Advanced Optical Character Recognition (OCR) via Google’s Tesseract OCR Engine
  • Single sign-on to major Cloud applications such as Dropbox, Box, Microsoft OneDrive for Business and Microsoft SharePoint Online
  • Type-ahead functionality to quickly filter the list of folders using the visually enhanced display at the MFP
  • Advanced metadata-to-file collection
  • Automated document renaming
  • Finger-swipe interface that makes scanning and uploading documents simple and secure.

The bizhub SECURE Notifier Dashboard provides a seamless integration between the bizhub SECURE Notifier app and any Dispatcher ScanTrip Cloud or Dispatcher Stratus tenant. Administrators can view an aggregate dashboard of all bizhub SECURE device statuses, set policies via a web interface, and configure automated as well as ad-hoc security reporting. Rather than manage the bizhub SECURE Notifier app settings and security status at each individual device, Dispatcher Stratus can communicate directly with the MFP, so administrators know the moment and MFP gets out of sync with the organization’s configured security standards.

Benefits

  • Visualization of Security Settings
  • Email Notifications

Shield Guard is an advanced cloud platform for remote, secure monitoring of your MFP fleet. With Shield Guard’s intuitive yet powerful features, admins can configure and monitor security policies remotely, via the cloud. Admins can quickly assess their overall MFP security via a centralized graphical Dashboard, ensuring that any security threats are immediately detected while saving both time and effort. The zero-footprint MFP Shield Guard Agent allows for continuous monitoring, eliminating the need for constant pinging or scheduling of assessments. Shield Guard features easy-to-use tools for policy and device management, timely security alerts, powerful password management features, and more, providing immediate peace of mind knowing your devices are protected.

Benefits

  • Intelligent Security Monitoring
  • Visibility to the Entire Fleet
  • Automatic Notifications

Multifunction printers (MFPs) often pose a security risk, as organizations may not be aware of data or document breaches when they occur. Unauthorized use of MFPs by authorized users is a significant source of data breaches, leaving organizations in the dark about who used the MFP, for what purpose, and when. BreachAlert is a unique document and data protection application exclusive to Konica Minolta devices. It provides real-time notifications of data breaches involving Konica Minolta MFPs, helping organizations comply with company, industry, and government privacy and data security requirements. With BreachAlert, organizations can track and monitor documents copied, printed, scanned, faxed, and emailed by individual users. With the cloud version, data collected by MFP Document Security processes is stored in in a hosted cloud data store.

In today’s digital age, data security is more critical than ever. BreachAlert ensures that your customers receive immediate alerts in case of potential data breaches, enabling them to take swift action to mitigate risks. 

Breach Alert Notifications 

  • Key word and key data breach triggers; easily and quickly user-configurable 
  • Automatic email notification of breach alerts to user-designated personnel (such as a Compliance Officer) 
  • ·Key data includes SSN, credit & debit card patterns, patient & student numbers, and much more 
  • Create breach and alert severity levels 

MFP Activity Archive 

  • Full-text or keyword searchable archive of all MFP documents printed, copied, scanned, emailed, and faxed 
  • Document auto-archiving and categorizing of all MFP activities 
  • Searchable by day, time, device, user, and much more 
  • Audit trail of all MFP activity for compliance requirement 

Easy Administration 

  • Easy to add and view connected MFPs 
  • Easy to add/modify key words, key data, and alerts 
  • Easy to respond to email breach alerts 
  • Set up notification groups based on roles/user security
