Dispatcher Paragon Cloud V2.0 is designed to eliminate the complexities associated with traditional on-premise print infrastructure and deployment headaches. This Software-as-a-Service (SaaS) secure pull print solution simplifies and modernizes print infrastructures by migrating them to the cloud.

Key Features:

  • Unified Print Driver and Queue: Dispatcher Paragon Cloud V2.0 offers a full-featured, single print driver with a single pull print queue for all users across all locations, supporting multiple vendors. This ensures seamless printing regardless of the printer brand.
  • Location-Aware Printing: The new location-aware platform (PC Client required) automatically discovers and installs printers available at the user's location and removes them when they leave. This feature enhances productivity by allowing users to print immediately without needing IT support.
  • Scan to Cloud Integration: For customers seeking scan-to-cloud apps like Google Drive, OneDrive, and more, Dispatcher Paragon Cloud V2.0 provides seamless user experience with single sign-on (SSO) capabilities to Dispatcher ScanTrip Cloud (purchased separately).

Benefits:

  • Enhanced IT and User Experience: The full-featured single print driver works across all models and brands. Once authenticated at a device, Paragon Cloud V2.0 formats the job, including all finishing options, and sends it to the chosen printer seamlessly.
  • Improved Productivity: Location-awareness allows users to access printers without installing devices or drivers. Available printers are automatically displayed, and those no longer available are removed, reducing post-installation support calls and accommodating the hybrid and mobile workforce.
  • Single Sign-On to Dispatcher ScanTrip Cloud: With SSO, customers can access popular cloud apps like Microsoft OneDrive, Dropbox, and SharePoint Online, along with features like full or zonal OCR, data extraction, intelligent routing based on metadata, and document renaming.

Benefits

  • Automated IT Work: Hassle-free printing with immediate end-user fulfillment.

  • Simple Deployment: Easy deployment via Konica Minolta MarketPlace.

Dispatcher Phoenix

Konica Minolta's Dispatcher Phoenix is a powerful application that can help any business save time by automating document image processing, printing, and routing tasks via customizable workflows. With a large variety of processing features, virtually everything is possible – from extracting data from documents, converting files to PDF or Office formats, applying annotations, and renaming files to routing documents to folders, FTP servers, MFPs, or e-mail recipients – and it's all fully automatic! Unique LiveFlo technology provides a real-time view of documents as they are being processed – a great way to identifying bottlenecks and making sure files will reach their correct destinations.

Dispatcher Phoenix provides busy offices with the convenience and flexibility they need. The application is highly scalable up to the largest enterprise environments. Dispatcher Phoenix includes a web user interface for access to important enterprise tools – such as apps for setting up server clusters for redundancy/load balancing, failover, offloading, sharing workflows with specific users, and more. Administrators can manage their workflows (run, stop, pause) from the web as well as edit user variables and view important analytics about work being done, including the number of documents being scanned, files collected, and users scanning.

Benefits

  • Multiple capturing methods.
  • Full integration with the MFP
  • Featuring drag-and-drop functionality, customisable icons, drawing tools, and easy-to-follow validation messages, the graphics-based Workflow Builder Tool makes creating workflows easier than ever.
  • Network-intensive processes can be scheduled to run after hours to maximize network performance and free up bandwidth during the workday
  • Time-consuming tasks can easily be automated, eliminating the need for manual intervention.
  • Dispatcher Phoenix comes with a variety of different connectors. SharePoint, Google Drive, Hyland OnBase are just some of the backend systems that allow bi-directional communication with Dispatcher Phoenix.
  • Offering output into databases as well as xml data, Dispatcher Phoenix grants connectivity to almost any backend system.
